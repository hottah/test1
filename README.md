## Overview
Home / About / Works の 3ページ構成のSPAです。  
**React + React Router** で実装し、**GitHub Pages** にホスティングしました。

## Tech Choices
- React: コンポーネント再利用と高速なDOM差分更新に絞って実装しました。
- React Router: SPAでURLに応じた表示切替をおこないます。
- GitHub Pages + gh-pages: 画面だけでなくコードの閲覧も可能とする選択になります。
