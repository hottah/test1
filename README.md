## Overview
Home / About / Works の 3ページ構成のSPA。  
**React + React Router** で実装し、**GitHub Pages** にホスティング。

## Tech Choices
- React: コンポーネント再利用と高速なDOM差分更新
- React Router: SPAでURLに応じた表示切替
- GitHub Pages + gh-pages: 無料・高速・更新が容易
